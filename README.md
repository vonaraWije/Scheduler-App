# Scheduler App

This is a simple scheduler application built for practice purposes.  
The app allows users to create, view, update, and delete scheduled appointments.  
It is designed to help a student understand basic full-stack development concepts such as CRUD operations, API handling, and database integration.

## Purpose
- Practice MERN stack development
- Learn REST API integration
- Improve frontend and backend connectivity

## Tech Stack
- MongoDB
- Express.js
- React
- Node.js
